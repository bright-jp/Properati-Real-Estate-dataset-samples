この**Properati 不動産データセットサンプル**には**11,961件のレコード**が含まれており、**Bright Data API**を使用して抽出されました。

## データポイント

| Field Name | Description |
|------------|-------------|
| `Type` | 物件の種類。例: Casa（戸建て）、Departamento（アパート）。 |
| `Name` | 物件掲載を説明する簡潔なタイトルまたは名称。 |
| `Estrato` | 物件エリアの社会経済的分類。 |
| `Habitaciones` | 物件の寝室数。 |
| `Banos` | 物件のバスルーム数。 |
| `M2` | 物件の総面積（平方メートル）。 |
| `Descripcion` | 設備やアメニティを含む物件の詳細説明。 |
| `Precio` | 物件の掲載価格。 |
| `Currency` | 物件価格が掲載されている通貨。例: USD、ARS。 |
| `Seller_name` | 物件を販売している個人または代理店の名前。 |
| `Publica_desde` | 物件掲載が最初に公開された日付。 |
| `Imagen` | 物件を表す画像URLのリスト。 |
| `Características` | 物件の主要な特徴および設備をJSON形式で表したもの。 |
| `Antiguedad` | 建築後の経過年数としての物件築年数。 |
| `Provincia` | 物件が所在する州または県。 |
| `Region` | 物件の州内における特定の地域または地区。 |
| `Address` | 物件の完全な住所。 |
| `Geolocation` | 物件の地理座標（緯度および経度）。 |
| `venta_arriendo` | 物件が売買用か賃貸用かを示します。 |
| `Phone` | 売主または掲載代理店の連絡先電話番号。 |
| `ID` | 物件掲載の一意の識別子。 |
| `url` | 物件掲載詳細ページへのURLリンク。 |
| `Seller_id` | 売主または掲載代理店の一意の識別子。 |
| `Active` | 掲載ステータス。現在アクティブか非アクティブかを示します。 |


## ユースケース

### 1. 不動産市場分析
このデータセットを活用して不動産市場のトレンドを分析し、アルゼンチンおよびコロンビアにおける物件価値と地域需要を評価します。

### 2. 投資機会
物件の特徴、価格、所在地を調査し、戦略的な意思決定のための潜在的な不動産投資機会を特定します。

### 3. 都市開発研究
政策立案や都市計画の取り組みを支援するために、都市開発パターンと住宅供給状況に関する研究を実施します。
